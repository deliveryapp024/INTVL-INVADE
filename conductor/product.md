# Initial Concept
Clone of the INTVL app (gamified running, territory capture, social features) specifically tailored for the Indian market. Mobile application for both iOS and Android.

## Key Features (MVP)
- **Activity Tracking:** Record runs with real-time metrics (duration, distance, pace).
- **Route Visualization:** View completed run routes on a minimalist, high-contrast map.
- **Territory Zones:** Compete for ownership of real-world areas using a hexagonal grid (H3). Capture zones by maintaining the highest distance in a weekly cycle.
- **Loop Master:** High-skill mechanic rewarding route enclosure with bonus captures.
