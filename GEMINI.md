# INVTL Workspace

## Project Overview
This directory serves as the root workspace for the **INVTL** project. Currently, it contains the VS Code workspace configuration and is ready for project initialization.

## Directory Structure
- `workspace.code-workspace`: Visual Studio Code workspace configuration file.
- `GEMINI.md`: Context file for the Gemini AI assistant.

## Usage
This directory is intended to house the source code and resources for the INVTL project. Future project files should be added here or linked via the workspace configuration.
